Lógica do espaçamento: O espaço entre grupos diferentes é maior do que o espaço entre itens do mesmo grupo. Isso cria uma separação visual clara sem a necessidade de desenhar linhas ou bordas.

Etapa 3 — Arquitetura CSS

No arquivo profile-card.html, nós aplicamos essa lógica na prática utilizando CSS Flexbox.

/* Definindo as variáveis para os espaçamentos */
:root {
  --espaco-pequeno: 6px;  /* Para itens próximos (mesmo grupo) */
  --espaco-grande: 20px;  /* Para separar as seções principais */
}

/* Container do conteúdo que separa os grupos */
.card-body {
  display: flex;
  flex-direction: column;
  gap: var(--espaco-grande); /* Cria a separação maior (20px) */
}

/* Cada grupo individual de informações */
.info-group {
  display: flex;
  flex-direction: column;
  gap: var(--espaco-pequeno); /* Cria o espaçamento menor (6px) */
}
Utilizamos a propriedade gap do Flexbox porque ela torna fácil controlar o espaço em branco apenas entre os itens. É uma abordagem mais limpa do que ter que calcular e aplicar margin-bottom de forma individual.
